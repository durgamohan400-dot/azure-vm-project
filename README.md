# 🚀 Azure VM Web Deployment Project

## 📌 Project Overview
This project demonstrates deploying a web application on an Azure Linux VM using Nginx and GitHub.

## 🛠️ Technologies Used
- Microsoft Azure (Virtual Machine)
- Nginx Web Server
- Git & GitHub
- Linux (Ubuntu)

## ⚙️ Steps Performed
1. Created Azure Linux VM
2. Configured Network Security Group (Port 80)
3. Installed and configured Nginx
4. Created and pushed code to GitHub
5. Deployed application on VM
6. Automated deployment using shell script
